 ENVIRONMENT SETUP

You'll need these 4 environment variables:

1. NEXT_PUBLIC_SUPABASE_URL
   └─ From Supabase Project Settings → API

2. NEXT_PUBLIC_SUPABASE_ANON_KEY
   └─ From Supabase Project Settings → API

3. SUPABASE_SERVICE_ROLE_KEY
   └─ From Supabase Project Settings → API

4. OPENAI_API_KEY
   └─ From OpenAI Platform → API Keys
